# Content Guidelines

## Tone

- Professional and calm
- Clear about uncertainty and limitations
- No official-service implication
- No hype or exaggerated certainty

## Mandatory disclaimer patterns

Include equivalent wording in visible UI and static pages:

- Unofficial personal/public project
- Not affiliated with Translink or Queensland Government
- Data may be delayed/incomplete/approximate
- Verify critical travel decisions with official sources

## Feature truthfulness

- Use "direct estimate" wording only for supported same-route direct journeys
- If unsupported, explicitly state transfer planning is not yet available
- Never imply fare, walking, or best-route optimization when absent

## Data/source pages

Before public launch, replace placeholders with:

- open data license text
- feed source URL(s)
- retrieval timing/frequency notes
- third-party asset attributions
