# Accessibility Notes

## Implemented baseline

- Semantic landmarks and headings on all pages
- Interactive controls use native buttons/inputs/selects
- Keyboard focus styles for links/buttons/fields
- Keyboard map/search shortcuts (`/`, `Cmd/Ctrl+K`)
- Search results navigable with arrow keys and Enter
- Contrast-safe color system for core text and controls

## Responsive behavior

- Desktop: map + side panel simultaneous view
- Mobile: expandable detail sheet to reduce crowding
- Route/stop list pages remain single-column friendly on small screens

## Known gaps / future improvements

- Full ARIA live-region tuning for all dynamic panel sections
- More explicit map keyboard navigation model
- Optional reduced-motion mode
- Additional colorblind-safe route palette presets
